# OmniFlow

OmniFlow是一个智能流程处理平台，它允许用户通过直观的可视化界面创建、编辑和执行自定义流程，并利用大语言模型(LLM)增强流程处理能力。

## 项目概述

OmniFlow提供了一套全面的工具，帮助用户：

- 通过直观的拖拽界面创建复杂的流程图
- 连接多种节点类型，实现自动化任务
- 集成大语言模型(如DeepSeek、GPT)进行高级文本处理
- 通过聊天界面与系统进行自然语言交互
- 管理和配置多种API密钥和模型参数

## 核心功能

### 流程编辑器

- 支持多种节点类型：输入节点、处理节点、决策节点、输出节点、API调用节点、LLM节点等
- 通过拖拽方式创建节点和连接
- 可定制节点属性和连接关系
- 支持流程验证、保存、加载和执行
- 提供迷你地图和各种视图控制选项

### 聊天界面

- 与大语言模型进行自然语言对话
- 支持多种消息类型：文本、问题、命令
- 可选择流程处理消息
- 自定义对话设置和模型参数

### 系统设置

- 配置多种LLM的API密钥(DeepSeek、OpenAI等)
- 选择默认使用的模型
- 调整模型参数(Temperature、最大Token数等)
- 测试API连接状态

## 技术架构

OmniFlow采用前后端分离的架构：

- 前端：HTML5, CSS3, JavaScript (使用Bootstrap框架)
- 后端：Python Flask提供Web服务和API
- 数据存储：JSON文件(流程定义、聊天记录、配置信息)
- 外部集成：支持多种LLM API(DeepSeek、OpenAI等)

## 快速开始

1. 克隆项目到本地
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行应用：
   ```bash
   python main.py
   ```
   或使用批处理文件：
   ```bash
   run.bat
   ```
4. 在浏览器中访问: `http://localhost:5000`

## 使用指南

### 流程创建

1. 访问"流程编辑器"页面
2. 从左侧节点库拖拽节点到画布
3. 连接节点的输入/输出点创建流程
4. 点击节点可编辑其属性
5. 保存流程以备后续使用

### 聊天交互

1. 访问"聊天界面"页面
2. 选择要使用的模型和参数
3. 输入消息或问题并发送
4. 可选择使用流程处理消息以获得更复杂的响应

### 系统配置

1. 访问"系统设置"页面
2. 输入各LLM提供商的API密钥
3. 测试连接状态
4. 配置默认模型和参数
5. 保存设置
